# FilmhubMusic-Back

## About

An online application that shows the user which popular albums are being streamed in Apple Music

## Technologies

- React
- Styled-components
- Navigation (Router)
- Axios

## How to run

1. Clone this repository
```bash
git clone https://github.com/rodnei0/FilmhubMusic-Back.git
```

2. Install dependencies
```bash
npm i
```

3. Run the app with
```bash
npm run dev
```
